# Jogo-clique-bolinha

https://felipe-gabriel-souza-goncalves.github.io/Jogo-clique-bolinha/index.html

Jogo simples feito em Javascript em que o jogador deve clicar na bolinha gerada
antes que o tempo (2seg) acabe para marcar pontos.

Toda vez que a bola for clicada o tempo será reiniciado para continuar jogando,
o jogo não possui final
